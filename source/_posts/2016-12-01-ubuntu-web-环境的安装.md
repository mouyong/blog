## 安装 web 服务器

### *二者选一即可*

#### 1. apache2

    sudo apt install apache2

#### 2. nginx

    sudo apt install nginx

## 安装数据库 mariadb

    sudo apt install mariadb-server

## 安装 php 以及相关扩展

    sudo apt install php php-zip  php-xml php-mbstring php-redis php-mysql php-gd
